# Bio-Morse Controller 🖐️

Proyecto para controlar el PC sin tocarlo, usando gestos y código Morse.

### ¿Qué va a hacer? (Ideas iniciales)

- Usar una cámara para leer movimientos de la mano.
- Mover el mouse y hacer clics (izquierdo/derecho).
- Interpretar código Morse con gestos para escribir texto.

### El reto:

Implementar y entrenar mi propia IA para que reconozca los gestos sin depender de APIs externas de texto. Resolver un problema que no existe de la manera más interesante posible.
